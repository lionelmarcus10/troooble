# Troooble

Cloud security challenge platform.

## Project Structure

```
troooble/
├── challenges/
├── challenge-orchestrator-service/
├── troooble-aws-network/
└── website/
```

| Folder | Description |
|--------|-------------|
| `challenges/` | Challenge definitions and configurations |
| `challenge-orchestrator-service/` | Backend API for deploying, validating and undeploying challenges |
| `troooble-aws-network/` | Infrastructure as Code (AWS) |
| `website/` | Next.js frontend with authentication and subscription management |

## Getting Started

See individual folder READMEs for setup instructions.
